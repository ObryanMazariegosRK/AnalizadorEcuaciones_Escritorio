# Analizador de Ecuaciones (Versión Escritorio) Desktop App

Aplicación de escritorio desarrollada en Java que permite analizar y resolver ecuaciones matemáticas (especialmente de segundo grado) de forma automática.

## Características

* **Análisis de cadenas:** Interpreta la ecuación ingresada en formato de texto (ej. `x^2-4=0`).
* **Cálculo de raíces:** Obtiene las soluciones utilizando la fórmula general.
* **Interfaz Gráfica:** UI amigable para el usuario (Java Swing/AWT).
* **Manejo de Errores:** Detecta sintaxis incorrecta o ecuaciones inválidas.

## Tecnologías Utilizadas

* **Lenguaje:** Java (JDK 8 o superior).
* **Entorno:** NetBeans IDE.
* **Gestor de construcción:** Apache Ant.
* **Librerías:** Incluidas en la carpeta `lib`.

## ¿Cómo ejecutar el proyecto?

Este proyecto fue creado con **NetBeans** y **Ant**.

1.  **Descargar:** Clona el repositorio o descarga el ZIP.
2.  **Abrir:** Inicia NetBeans, ve a `File > Open Project` y selecciona la carpeta descargada.
3.  **Construir:** Haz clic derecho en el proyecto y selecciona `Clean and Build`.
4.  **Ejecutar:** Haz clic en el botón de `Run` (Play) o presiona `F6`.
